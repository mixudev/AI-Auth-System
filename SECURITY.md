# Security Policy

## Supported Versions

Sistem ini sangat mementingkan keamanan data pengguna. Kami menyarankan untuk selalu menggunakan versi terbaru dari MixuAuth.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

Kami sangat menghargai bantuan Anda dalam menjaga keamanan MixuAuth. Jika Anda menemukan celah keamanan, mohon ikuti langkah berikut:

1. **Jangan** memposting isu keamanan secara publik (seperti di GitHub Issues).
2. Kirim detail temuan Anda ke email: **security@mixudev.id**
3. Kami akan merespons dalam waktu 24-48 jam dan bekerja sama dengan Anda untuk proses patching.

Setelah patch dirilis, kami akan memberikan kredit kepada penemu celah keamanan di dokumen kontributor kami.
