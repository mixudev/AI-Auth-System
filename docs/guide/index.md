# Ringkasan Panduan

Bagian ini adalah entry point untuk tim developer, DevOps, dan QA.

## Tujuan Dokumen Guide

- Menjalankan sistem dari nol.
- Memahami relasi antar service Docker.
- Mengelola konfigurasi `.env` tanpa salah langkah.
- Menjalankan operasional harian dengan runbook yang jelas.
- Menangani error umum dengan langkah troubleshooting terstruktur.
